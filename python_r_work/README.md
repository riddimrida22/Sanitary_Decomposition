# python_r_work

This folder contains one Python script and one R script in `src/`.

## Prerequisites
- Python 3
- R (or `Rscript`)

## Run the scripts
From the repository root:

```bash
python3 python_r_work/src/hello.py
Rscript python_r_work/src/hello.R
```

Expected output:
- Python: `Hello from Python in python_r_work/src`
- R: `Hello from R in python_r_work/src`
